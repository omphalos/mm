mm
==

Simple text-based non-hierarchical mind-mapping utility.

QuickStart
==========

Start the REPL:

    ./mm.js
    
Create an entry:

    add lotr-characters

Navigate to it:

    goto lotr-characters
    
Add some entries:

    add gandalf
    add frodo
    
View your work:

    query

Save your work:

    save

At any time, get a full list of commands with:

    help
